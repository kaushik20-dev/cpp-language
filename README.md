# C++ Language
Subset C++ Practice codes of Google C++ class (https://developers.google.com/edu/c++)

## Windows 10
### Compile C++ program
```
g++ -std=c++14 -o program.exe program.cpp
```
### Run C++ program
```
program.exe
```
## Mac OS
### Compile C++ program
```
g++ -std=c++14 -o program.o program.cpp
```
### Run C++ program
```
./program.o
```
